	При включении появится окно авторизации. Приложение использует подключение к удаленной базе данных находящейся на арендованном хостинге. В приложении реализована авторизация только одного пользователя, т.е один логин и пароль используются для всех копий приложения. Вы можете изменить входные данные. Для этого нажмите Option - Admin и введите новые данные. Меняя логин и пароль в своем клиенте, вы меняете логин и пароль для всех остальных копий приложения. 
	Последние входные данные - Login: Stalker, Pass: Zona
	В случае некорректных данных Вы можете воспользоваться восстановлением данных через соответствующее окно. На Вашу электронную почту придет сообщение с данными входа. 
	Имеется функция выхода из учетной записи: File - Log out.


Персонажи классифицированны по фракциям, поэтому сначала необходимо выбрать фракцию, потом выбрать из списка нужного персонажа.

Карточка персонажа позволяет:
	- Получить необходимую информацию о персонаже.
	- Отредактировати и сохранить информацию о персонаже.
	- Удалить информационную карточку персонажа.
	- Связаться с персонажем путем отправки сообщения по указанному электронному адресу.

Для регистрации нового персонажа откройте форму регистрации (File - New Person) и введите необходимые данные.
