На Вашу электронную почту придет сообщение с данными входа.
Вы можете изменить входные данные. Для этого нажмите Option - Admin и введите новые данные.

Персонажи классифицированны по фракциям, поэтому сначала необходимо выбрать фракцию, потом выбрать из списка нужного персонажа.

Карточка персонажа позволяет:
	- Получить необходимую информацию о персонаже.
	- Отредактировати и сохранить информацию о персонаже.
	- Удалить информационную карточку персонажа.
	- Связаться с персонажем путем отправки сообщения по указанному электронному адресу.

Для регистрации нового персонажа откройте форму регистрации (File - New Person) и введите необходимые данные.

Имеется функция выхода из учетной записи: File - Log out.
